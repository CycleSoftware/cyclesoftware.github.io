    Scope(s)

    • resources

    Customer (object)

    Property Type Nullable Modify Description
    customer_id integer false readonly Customer ID e.g. 6
    customer_type_name string false PUT|POST See common enum customer_types e.g. Klant
    customer_reference string false PUT|POST Reference of customer e.g. OwnRef
    postcode string false PUT|POST Postcode of customer e.g. 5488HM
    house_number string false PUT|POST House number of customer e.g. 1
    house_number_postfix string false PUT|POST House number postfix of customer e.g. c
    attn string false PUT|POST Attn name Name of address
    title string false PUT|POST Name prefix e.g. Dhr./Mevr.
    initials string false PUT|POST Initials e.g. T
    insertion string false PUT|POST Insertion e.g. van
    first_name string false PUT|POST First name e.g. Tim
    name string false PUT|POST Last name e.g. LastName
    street string false PUT|POST Street e.g. Raadhuisplein
    city string false PUT|POST City e.g. Nistelrode
    country_code_iso_3166 string false PUT|POST Country code e.g. NL
    email string false PUT|POST E-mail e.g. name@domain.nl
    discount_percentage decimal false PUT|POST Default discount percentage e.g. 10
    datetime_created datetime true readonly Created at if available e.g. 2023-01-03 12:30:44
    phone_numbers array false PUT|POST array of phone numbers
    phone_numbers[].phone_number_id string false PUT|POST ID of phone number e.g. tel (landline) mob (mobile) or ID
    phone_numbers[].customer_id integer false readonly Customer ID e.g. 6
    phone_numbers[].phone_number string false PUT|POST Phone number e.g. 0733030050
    phone_numbers[].name string false PUT|POST Name of number e.g. Dhr./Mevr. T UnitTest

    Repair objects

    Read, create or update repair objects for a customer

    Scope(s)

    • resources

    Repair object (object)

    Property Type Modify Description
    customer_id integer readonly Customer ID of owner e.g. 6
    object_id integer readonly ID of the object e.g. 208410
    object_barcode string PUT|POST e.g. object_barcode
    is_active boolean PUT|POST Whether this object is still owned by customer e.g. true
    object_type_name string PUT|POST e.g. fiets
    category string PUT|POST Category name e.g. Racefietsen
    brand string PUT|POST Brand name e.g. BrandName
    model string PUT|POST Model name e.g. ModelName
    model_year string PUT|POST Model year e.g. 2017
    color string PUT|POST Color of object e.g. Black
    variant string PUT|POST Variant name e.g. Heren
    phone_number_id string PUT|POST Phone number ID for communication e.g. 12345
    license_plate string PUT|POST License plate number e.g. xx-zz-yy
    km_mileage string PUT|POST Kilometer age e.g. 200
    frame_id string PUT|POST Frame number e.g. frame_id
    chip_id string PUT|POST Chip number e.g. chip_id
    key_id string PUT|POST Key number e.g. key_id
    engine_id string PUT|POST Engine number e.g. engine_id
    battery_id string PUT|POST Battery number e.g. 1234
    lock_id string PUT|POST Lock number e.g. 1234
    workshop_rate_id integer PUT|POST See common API workshop_rates e.g. 1
    service_level_id integer PUT|POST Service level ID e.g. 0
    images array readonly Array of images
    images[].date_modified datetime|null readonly Modification date e.g. 2016-05-24 11:15:02
    images[].url_thumb string readonly URL to thumbnail e.g. https://s01.cyclesoftware.nl/app/img/artPic_public_T_1317089.jpg
    images[].url_large string readonly URL to large image e.g. https://s01.cyclesoftware.nl/app/img/artPic_public_L_1317089.jpg
